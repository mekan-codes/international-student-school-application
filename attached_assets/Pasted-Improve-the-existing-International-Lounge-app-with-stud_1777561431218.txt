2. STUDENT SELF-PICKUP FROM SUBSTITUTE FOOD LOCKER
==================================================

Currently students can only view available locker food.
Allow substitute-food students to take food online directly from the Substitute Food Locker page.

Rules:
- Only students with substitute-food access can use this page.
- Standard students must still not see or access this page.
- Students cannot take more than available locker quantity.
- Students cannot take zero or negative quantity.
- If locker quantity becomes low, existing admin low-stock alerts should still work.
- Every student pickup must update locker quantity.
- Every student pickup must create inventory history/log entries.
- Every student pickup must also appear in the Shareable Food Log / distribution log.

Preferred UX:
- Show available foods as cards or rows.
- Each available food has a quantity input.
- Use one main button:
  Take selected food

This allows a student to take multiple foods in one action.

Example:
Student selects:
- Chicken Mayo: 1
- Noodles: 1
- Rice: 1

After clicking Take selected food:
- locker quantities decrease
- one grouped pickup/distribution record is created
- inventory log entries are created for each food item
- success message:
  You took Chicken Mayo × 1, Noodles × 1, Rice × 1.

Grouped Shareable Food Log behavior:
If a student takes multiple foods in one action, show it as one grouped pickup row, not three separate pickup sessions.

Example:
April 29, 2026 | Mekan | Chicken Mayo: 1, Noodles: 1, Rice: 1

This should match the existing shareable food log format if possible.
