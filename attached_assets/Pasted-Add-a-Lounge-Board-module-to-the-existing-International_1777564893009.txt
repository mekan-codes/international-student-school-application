2. ADD LOUNGE BOARD MODULE
==================================================

Purpose:
Lounge Board is a public community space for international students and staff.

It is different from:
- Announcements = official staff notices
- Intl Dept. Requests = private support tickets to the international department
- Lounge Board = public community posts, comments, and reactions

Roles:
All logged-in users:
- view posts
- create posts
- comment on posts
- react to posts
- edit their own posts
- delete their own posts
- edit their own comments
- delete their own comments

Admin/manager:
- delete any inappropriate post/comment
- pin/unpin posts
- lock/unlock posts
- view all posts

3. RULES AND PERMISSIONS
==================================================

Post rules:
- title is required
- content is required
- category is required
- category defaults to General if needed
- users can edit/delete only their own posts
- admins/managers can delete any post
- admins/managers can pin/unpin any post
- admins/managers can lock/unlock any post

Comment rules:
- comment content is required
- users can edit/delete only their own comments
- admins/managers can delete any comment
- locked posts cannot receive new comments from normal users
- admins/managers may still comment on locked posts if simple

Reaction rules:
- each user can have only one reaction per post
- clicking a different reaction changes their reaction
- clicking the same reaction again removes it
- show reaction counts under posts
- if possible, update reactions without full page refresh using fetch/AJAX
- if AJAX is risky, normal form submit with refresh is acceptable

Privacy rules:
- do not show hidden phone numbers
- do not expose internal role/program labels to students
- do not show private profile fields
