3. CLEANING TASK STATUS CORRECTION
==================================================

Admin/manager should be able to change a task status if needed.

Allowed staff task status changes:
- assigned
- marked_done
- verified_done
- missed

Rules:
- Students can mark tasks done only if the session is active/scheduled and their team is assigned.
- Students cannot verify tasks.
- Students cannot change missed/verified/cancelled tasks.
- Admin/manager can reset missed task back to assigned if needed.
- Admin/manager can verify a marked_done task.
- Admin/manager can mark any non-deleted task missed.
- Admin/manager can delete a task if needed.

If the session is cancelled:
- students cannot mark tasks done
- admin/manager can still inspect/manage records where appropriate, but avoid accidental changes unless explicitly using Manage

If the session is approved:
- students cannot change tasks anymore
